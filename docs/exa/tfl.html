Usage Example - Operation: 'TF L'

Transfer Function - Generate CTF, in doc file

Procedure used: ./images/tfl.spi

.OPERATION: TF L ;   Generate CTF, in doc file
.SPHERICAL ABBERATION CS [mm]: 2.0 ;   CS
.DEFOCUS [A], ELECTRON VOLTAGE [Kev]: 48600, 300 ;   Defocus, electon voltage
.NUMBER OF SPATIAL FREQUENCY POINTS: 360 ;   Number of spatial freq. points
.MAXIMUM SPATIAL FREQUENCY [1/A]: 0.373 ;   Max spatial freq.
.SOURCE SIZE [1/A], DEFOCUS SPREAD [A]: 0.0, 0.0 ;   Source size, defocus spread
.AMPLITUDE CONTRAST RATIO [0-1], GAUSSIAN ENVELOPE HALFWIDTH: 0.1, 10000 ;   ACR, GEH (10000 = has no effect)
.DIFFRACTOGRAM, ENVELOPE, OR STRAIGHT (D/E/S): S ;   Diffractogram, Envelope, or Straight
.OUTPUT DOC FILE: tfl_doc ;   Doc file (output)
