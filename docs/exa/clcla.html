Usage Example - Operation: 'CL CLA'

Create Dendrogram from CORAN or PCA Analysis

Procedure used: ca.spi

See: Classification and Clustering Tutorial for further info.

.OPERATION: CL CLA ;   Create Dendrogram from CORAN or PCA Analysis
.CORAN/PCA FILE PREFIX: cas_c ;   IMC file prefix (input)
.CLUSTER OUTPUT FILE FOR 'CA SM' USE: clcla_clu ;   Cluster file (output)
.FACTOR NUMBERS TO BE USED: 1-3 ;   Factors used: 1, 2, 3
.NUMBER OF ITERATIONS & CENTERS PER GROUP: 5, 4 ;   Iterations and centers
.NUMBER OF INITIAL CLUSTERS: 8 ;   Number of starting clusters
.OCCUPANCY LEVEL FOR CLASS CUTOFF (0=NO CUTOFF): 0 ;   No cutoff for any class
.DENDROGRAM POSTSCRIPT OUTPUT FILE: clcla_den.ps ;   Dendrogram plot (output)
.PLOT CUTOFF (0...100): 2 ;   Dendrogram plot cutoff
.DENDROGRAM DOC. FILE: clcla_den_doc ;   Dendrogram doc file (output)
