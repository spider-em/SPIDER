Usage Example - Operation: 'DN MED'

Denoise using median filtering

Procedure used: dnmed.spi

.OPERATION: DN MED ;   Denoise using anisotropic diffusion
.INPUT FILE: sav_pp ;   File name (input)
.OUTPUT FILE: dnmed ;   File name (input)
.FILTER LENGTH 5 ;   Filter length
.BOX OR CROSS (B/C) c ;   Use cross shaped neighborhood


INPUT IMAGE OUTPUT IMAGE
sav_pp dnmed
