Usage Example - Operation: 'MK'

Marker based alignment - single tilt images

Procedure used: mk.spi

.OPERATION: MK ;   Marker based alignment
.PREFIX OF MARKER DOC FILES: sav_mk_doc_*** ;   Marker file prefix (input )
.FILE NUMBERS: 1-75 ;   File number range
.REFERENCE FILE NUMBER: 38 ;   File number of zero-tilt image
.X,Y IMAGE DIMENSIONS: 1024, 1024 ;   Size of image
.REFINE SCALE (VS REFERENCE)? (Y/N): Yes ;   Refine scale (vs reference)
.REFINE TILT ANGE (VS REFERENCE)? (Y/N): No ;   Refine tilt angle (vs reference)
.REFINE IN-PLANE ROTATION? (y/N): Yes ;   Refine in-plane rotation
.REFINE SHIFT (VS REFERENCE)? (Y/N): No ;   Refine shift (vs reference)
.USE PREVIOUS CORRECTIONS? (Y/N): No ;   Use previous corrections
.FULL OUTPUT? (Y/N): No ;   Do not want full output
.ERROR PER VIEW OUTPUT DOC FILE: mk_err_per_view_doc ;   Error per view output doc file (output)
.ERROR PER MARKER OUTPUT INPUT DOC FILE: mk_err_per_marker_doc;   Error per marker output doc file (output)
.CORRECTIONS OUTPUT DOC FILE: mk_corr_doc ;   Corrections output doc. file (output)
.3-D COORDS OUTPUT DOC FILE: mk_coord_doc ;   3-D coords output doc. file (output)
