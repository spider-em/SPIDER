Usage Example - Operation: 'ER'

Erode object in a binary image/volume

Procedure used: images/er.spi

.OPERATION: ER ;   Erode object to skeleton
.INPUT FILE: sav_bin ;   Image (input)
.OUTPUT FILE: er_C38 ;   Image (output)
.BOX OR CROSS NEIGHBORHOOD (B/C): B ;   Box shaped
.LENGTH OF NEIGHBORHOOD: 3 ;   3 pixels wide/high
.ERODE IF NUMBER OF OCCUPIED NEIGHBORS IS LESS THAN: 26 ;   Neighbors
