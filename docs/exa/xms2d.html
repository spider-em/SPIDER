Usage Example - Operation: 'XM S2D'

Convert XMipp star file to SPIDER doc file

Procedure used: xms2d.spi

.OPERATION: XM S2D ;   Convert XMipp star file to SPIDER doc file
.STAR FILE: star_1.star ;   Star file (input)
.STAR FILE REGISTER VARIABLES WANTED: _rlnOriginY,_rlnAngleRot,_rlnAngleTilt ;   Star variables wanted
.CORRESPONDING SPIDER DOC REGISTER NUMBERS: 1,2,3 ;   SPIDER doc file registers
.STAR FILE NAME VARIABLES WANTED (e.g. _rlnImageName): * ;   Star variable name containing filenames with integers inside
.SPIDER DOC FILE: xms2d_doc ;   SPIDER doc file (output)


STAR FILE INPUT
star_1


SPIDER DOC FILE OUTPUT
Xms2d_doc
