Usage Example - Operation: 'FQ'

Filter - Quick Fourier, high, low pass, etc., padded

Procedure used: fq.spi

.OPERATION: FQ ;   Quick Fourier filter
.INPUT FILE: sav_pp ;   Image (input)
.OUTPUT FILE: fq ;   Image (output)
.FILTER TYPE (1-10): 2 ;   Filter type
.FILTER RADIUS (IN FREQUENCY OR PIXEL UNITS): 10 ;   Radius


INPUT IMAGE OUTPUT IMAGE
sav_pp fq
