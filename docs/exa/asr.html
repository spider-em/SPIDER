Usage Example - Operation: 'AS R'

Average image/volume series, optional subset

Procedure used: asr.spi

.OPERATION: AS R ;   Average Images - Find Variance
.INPUT FILE TEMPLATE: sav_lowpass_pp_stk@*** ;   Prefix of images (input)
.FILE NUMBERS OR SELECT DOC. FILE NAME: 1-8 ;   Numbers of file names in the series
.ALL, ODD-EVEN (A/O): A ;   All, or odd/even averages calculated
.AVERAGE FILE: asr_avg ;   Average file (output)
.VARIANCE FILE: asr_var ;   Variance file (output)


INPUT FILES       AVERAGE FILE       VARIANCE FILE
           
sav_lowpass_pp_stk_mon       asr_avg       asr_var
