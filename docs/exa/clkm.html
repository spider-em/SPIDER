Usage Example - Operation: 'CL KM'

K means classification

Procedure used: clkm.spi

See: Classification and Clustering Tutorial for further info.

.OPERATION: CL KM ;   K means classification
.CORAN/PCA FILE: cas_c_IMC ;   Which file i.e.: SEQ, IMC, or PIX (input)
.NUMBER OF CLASSES: 8 ;   Number of classes
.FACTOR NUMBERS: 1-3 ;   Which factors
.FACTOR WEIGHT: 0 ;   Equal factor weight
.FOR RANDOM SEEDS GIVE NON-ZERO STARTING NUMBER: 453 ;   Seed
.SELECTION DOC FILE TEMPLATE: clkm_imc_clas_doc_*** ;   Selection doc files (output)
.CLASS MEMBERSHIP DOC FILE: clkm_imc_clas_doc ;   Class membership doc file (output)


INPUT TEXT FILE
(Truncated)
cas_c_IMC


SAMPLE CLASS SELECTION DOC FILE
(One of eight)
doc_clhc_den_2_doc_001


OUTPUT CLASS MEMBERSHIP DOC FILE
clkm_imc_clas_doc
