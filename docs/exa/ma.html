Usage Example - Operation: 'MA'

Mask image/volume using circular/spherical mask

Procedure used: ma.spi

.OPERATION: MA ;   Mask image
.INPUT FILE: sav_pp ;   Image name (input)
.OUTPUT FILE: ma ;   Image name (output)
.OUTER & INNER RADII: 32,0 ;   Radii
.DISK, COSINE, GAUSSIAN EDGE, OR TRUE GAUSSIAN: D ;   Sharp disk
.AVG, PREV AVG, CIRCUMFERANCE, OR EXTERNAL BACKGROUND: Aver ;   Average
.MASK CENTER LOCATION (X,Y) OR <CR&gr; FOR IMAGE CENTER: 50, 50 ;   Mask center


INPUT IMAGE OUTPUT IMAGE
sav_pp ma
