Usage Example - Operation: 'TM'

Record elapsed time

Procedure used: ./images/tm.spi

.OPERATION: TM ;   Reset time to zero
.  
.OPERATION: SYS ;   Sleep 2 seconds
.INPUT FILE: sleep 2 ;  
.  
.OPERATION: TM [elapsed] ;   Record elapsed time since last 'TM'
.  
.OPERATION: SYS ;   Echo elapsed time
.SYSTEM COMMAND: echo " Elapsed time: {%f6.3%[elapsed]}"
.
