Usage Example - Operation: 'DOC SHUFFLE'

Shuffle Document File

Procedure used: docshuffle.spi

.OPERATION: DOC SHUFFLE ;   Shuffle document file
.SOURCE DOCUMENT FILE: sav_doc_10 ;   Doc file (input)
.FIRST & LAST KEY NUMBER: 1,20 ;   First and last keys to use
.INPUT REGISTER # FOR OUTPUT REG. 2: 1 ;   Reg. number of value stored in register 2
.INPUT REGISTER #s FOR OUTPUT REG. 3 AND 4: 1,1 ;   Reg. numbers of values stored in register 3 and 4
.DOCUMENT FILE: docshuffle ;   Shuffled doc file (output)


INPUT DOC. FILE OUTPUT DOC. FILE
sav_doc_10 docshuffle
