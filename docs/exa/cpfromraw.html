Usage Example - Operation: 'CP FROM RAW'

Copy 8 bit RAW file to SPIDER file

Procedure used: cpfromraw.spi

.OPERATION: CP TO RAW ;   Copy SPIDER file To RAW file
.SPIDER INPUT FILE: savpad_nik_sli_45 ;   SPIDER image (input)
.NORMALIZE THE OUTPUT TO 0-255? (Y/N): Yes ;   Normalize 0...255
.PAD TO INCREMENT OF 4 IF NECESSARY?(Y/N): No ;   Do not pad
.EIGHT BIT OUTPUT FILE: cptoraw_2 ;   RAW image (output)
. ;  
.OPERATION: CP FROM RAW ;   Create SPIDER image from MRC file
.EXISTING RAW FILE: cptoraw_2 ;   MRC image (input)
.BITS/PIXEL IN INPUT IMAGE(8, 16, 32 or -32): 8 ;   SPIDER image (output)
.NX, NY, & NZ: 87, 87, 1 ;   Image size
.HEADER BYTES TO BE SKIPPED: 0 ;   Header bytes in raw file
.NEW SPIDER IMAGE FILE: cptofromraw ;   SPIDER image (input)


INPUT IMAGE
savpad_nik_sli_45

OUTPUT IMAGE
cptofromraw

RESULTS FILE
