Usage Example - Operation: 'OR SH'

Orientation Search - 2D, rotation, uses polar coords

Procedure used: orsh.spi

.OPERATION: OR SH [ang],[x],[y],[m],[cc] ;   Orientation Search
.REFERENCE IMAGE: sav_avg000 ;   Reference image (input)
.TRANSLATION SEARCH RANGE IN X & Y, STEP:5, 5, 1 ;   Translation shifts, shift step
.FIRST, LAST RING: 5, 25, 1 ;   Search radii, radius step
.EXPERIMENTAL IMAGE: sav_avg_315 ;   Experimental image (input)
.CHECK MIRRORED POSITIONS? (Y/N): Y ;   Check mirrors


Exp. Image Ref. image
sav_avg000 sav_avg_315


Only output is to operation line registers:

[ang] [x] [y] [mir] [cc]
 45.01   0.0   0.0   0.0  2065.16
