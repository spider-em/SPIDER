Usage Example - Stack creation operations

Procedure used: ./images/ms.spi

.OPERATION: MS ;   Create inline image stack
.NEW INLINE STACK FILE: _1 ;   Inline stack number 1 (output)
.X, Y, & Z DIMENSIONS: 75,75,1 ;   Size of images
.NUMBER OF IMAGES/VOLUMES ALLOWED: 10 ;   Number of images allowed in stack
.
.OPERATION: MS 3 ;   Create inline volume stack
.NEW INLINE STACK FILE: _2 ;   Inline stack number 2 (output)
.X, Y, & Z DIMENSIONS: 75,75,3 ;   Size of volumes
.NUMBER OF IMAGES/VOLUMES ALLOWED: 5 ;   Number of volumes allowed in stack
.
.OPERATION: MS F ;   Create inline Fourier image stack
.NEW INLINE STACK FILE: _3 ;   Inline stack number 3 (output)
.X, Y, & Z DIMENSIONS: 75,75,1 ;   Size of images
.NUMBER OF IMAGES/VOLUMES ALLOWED: 10 ;   Number of images allowed in stack
.
.OPERATION: MS I ;   Create inline indexed image stack
.NEW INLINE STACK FILE: _4 ;   Inline stack number 4 (output)
.X, Y, & Z DIMENSIONS: 75,75,1 ;   Size of images
.NUMBER OF IMAGES/VOLUMES ALLOWED: 10 ;   Number of images allowed in stack
.HIGHEST IMAGE/VOLUME NUMBER ALLOWED IN STACK: 100 ;   Highest image number allowed in stack
.
.OPERATION: MS I ;   Create disk based image stack
.NEW INLINE STACK FILE: jnk_ind_stk ;   Disk based stack name (output)
.X, Y, & Z DIMENSIONS: 75,75,1 ;   Size of images
.NUMBER OF IMAGES/VOLUMES ALLOWED: 10 ;   Number of images allowed in stack
.
