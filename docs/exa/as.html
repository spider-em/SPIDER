Usage Example - Operation: 'AS'

Average Images - Find Variance

Procedure used: as.spi

.OPERATION: AS ;   Average Images - Find Variance
.INPUT FILE TEMPLATE: sav_lowpass_pp_stk@*** ;   Prefix of images (input)
.FILE NUMBERS OR SELECT DOC. FILE NAME: 1-8 ;   Numbers of file names in the series
.ALL, ODD-EVEN (A/O): A ;   All, or odd/even averages calculated
.AVERAGE FILE: as_avg ;   Name of Average file (output)
.VARIANCE FILE: as_var ;   Name of Variance file (output)


INPUT FILES       OUTPUT AVERAGE FILE OUTPUT VARIANCE FILE
     
sav_lowpass_pp_stk_mon       as_avg as_var
