Usage Example - Operation: 'IQ PAR'

Stops SPIDER if a string variable does not hold a specified value.

Procedure used: iqpar.spi

.OPERATION IQ PAR ;   Inquire if string variable holds specified value
.VARIABLE NAME (WITH[]): [str] ;   String variable name
.CORRECT VALUE: cow ;   String variable contents
