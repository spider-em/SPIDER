Usage Example - Operation: 'TF CT3'

Transfer Function - Generate a binary, phase flipping, complex, CTF correction volume

Procedure used: ./images/tfct3.spi

.OPERATION: TF CT3 ;   Generate Transfer Function - Complex 3D
.CTF OUTPUT FILE: tfct3 ;   Complex CTF correction volume (output)
.SPHERICAL ABBERATION CS [mm]: 2.0 ;   CS
.DEFOCUS [A], ELECTRON VOLTAGE [Kev]: 3660, 300 ;   Defocus, electon voltage
.NUMBER OF SPATIAL FREQUENCY POINTS: 64 ;   Number of spatial freq. points
.MAXIMUM SPATIAL FREQUENCY [1/A]: 0.427 ;   Max spatial freq.
.SOURCE SIZE [1/A], DEFOCUS SPREAD [A]: 0.0, 0.0 ;   Source size, defocus spread
.ASTIGMATISM [A], AZIMUTH [DEG]: 2000, 0 ;   Astigmatism, azimuth
.AMPLITUDE CONTRAST RATIO [0-1]: 0.1 ;   ACR )
.SIGN [+1 or -1]: -1 ;   Sign


OUTPUT FILE
(binary 3D, Not displayable)
tfct3
