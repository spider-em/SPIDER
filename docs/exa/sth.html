Usage Example - Operation: 'ST H'

Set values in image/volume Header by name

Procedure used: sth.spi

.OPERATION: ST H ;   Set values in image header
.SET HEADER VALUES IN FILE: sav_st ;   Image (output)
.HEADER VARIABLE(S): PROJ,MIC ;   Header locations
.VALUE(S): 5, 2102 ;   Header locations
.  
.OPERATION: FI H [proj],[mic] ;   Get values from image header
.INPUT FILE: sav_st ;   Image (input)
.HEADER VARIABLE(S): PROJ,MIC ;   Header locations
.  
.OPERATION: SYS ;  
.SYSTEM COMMAND: echo ' Project: {**[proj]} Micrograph: {*****[mic]}' ;
