Usage Example - Operation: 'AD F'

Add Two Images with Factors

Procedure used: adf.spi

.OPERATION: AD F ;   Add images with factors
.INPUT FILE: sav_rot_ax_0 ;   First image to be added
.NEXT INPUT FILE: sav_rot_ax_30 ;   Second image to be added
.FACTORS FOR FIRST & SECOND FILES: 5,10 ;   Factors
.OUTPUT FILE: adf ;   Added image name


INPUT NEXT INPUT OUTPUT
sav_rot_ax_0 sav_rot_ax_30 adf
