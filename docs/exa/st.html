Usage Example - Operation: 'ST'

Set values in image/volume header

Procedure used: st.spi

.OPERATION: ST ;   Set values in image header
.INPUT FILE: sav_st ;   Image (output)
.(A)NGLES, (B)UF, (P)COPY, (C)LEAR, OR (F)IX (A/B/P/C/F):A ;   Alter angles
.ANGLE SET 1, 2, OR 3: 1 ;   Use first angle set
.PHI, THETA, & PSI: 30, 60, 90 ;   Angles
.  
.OPERATION: FI H [phi],[the] ;   Get values from image header
.INPUT FILE: sav_st ;   Image (input)
.HEADER VARIABLE(S): PHI,THETA ;   Header locations
.  
.OPERATION: SYS ;  
.SYSTEM COMMAND: echo ' Phi: {%f7.2%[phi]}" " Theta: {%f7.2%[the]}' ;
