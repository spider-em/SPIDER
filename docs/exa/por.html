Usage Example - Operation: 'PO R'

Convert Image to Polar Coordinate Representation - rays along X

Procedure used: por.spi

.OPERATION: PO R ;   Convert to polar coordinates representation
.INPUT FILE: sav_pp ;   File name (input
.OUTPUT FILE: por ;   File name (output)
.FULL OR HALF CIRCLE (F/H): Full ;   Convert Full or Half circle


INPUT OUTPUT
sav_pp por
