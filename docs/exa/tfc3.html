Usage Example - Operation: 'TF C3'

Generate Transfer Function - Complex 3D

Procedure used: ./images/tfc3.spi

.OPERATION: TF C3 ;   Generate Transfer Function - Complex 3D
.OUTPUT FILE: tfc3 ;   Complex 3D file (output)
.SPHERICAL ABBERATION CS [MM]: 2.0 ;   CS
.DEFOCUS [], LAMBDA [A]: 3660, 0.0197 ;   Defocus , lambda
.DIMENSIONS OF OUTPUT ARRAY: 64 ;   Number of spatial frequency points
.MAXIMUM SPATIAL FREQUENCY [A-1]: 0.427 ;   Maximum spatial frequency
.SOURCE SIZE [A-1], DEFOCUS SPREAD [A]: 0.005, 0.0 ;   Source size , defocus spread
.ASTIGMATISM [M], AZIMUTH[DEG]: 2000,0 ;   Astigmatism, azimuth
.AMPLITUDE CONTRAST RATIO [0-1], GAUSSIAN ENVELOPE HALFWIDTH [1/A]: 0.1 , 10000 ;   ACR, GEH
.SIGN[+1 or -1]: -1 ;   Sign


OUTPUT FILE
(Complex 3D)
tfc3
