Usage Example - Operation: 'PJ 2'

Project image onto line

Procedure used: pj2.spi

.OPERATION: PJ 2 ;   Project image onto line
.INPUT FILE: sav_pj2 ;   Imag (input)
.PROJECT ALONG X or Y AXIS? (X/Y): Y ;   Projection axis
.OUTPUT FILE: pj2y ;   Image (output)

.OPERATION: PJ 2 ;   Project image onto line

.OUTPUT FILE: pj2x ;   Image (output)
