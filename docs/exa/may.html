Usage Example - Operation: 'MA Y'

Mask - Image in Y Direction

Procedure used: may.spi

.OPERATION: MA Y ;   Mask - image in Y direction
.INPUT FILE: sav_pp ;   File name (input)
.OUTPUT FILE: may ;   File name (output)
.OUTER & INNER HALFWIDTH EXTENTS: 0, 10 ;   Outer & inner halfwidth extents of mask
.DISK, COSINE, GAUSSIAN EDGE OR TRUE GAUSSIAN (D/C/G/T): COSINE ;   Mask type
.AVG, PREV AVG, CIRCUMF, OR EXTERNAL BACKGROUND? (A/P/C/E): AVG ;   Background type
.Y CENTER OF MASK OR <CR&gr; FOR IMAGE CENTER: 30 ;   Outer & innter halfwidth extents of mask


INPUT IMAGE OUTPUT IMAGE
sav_pp may
