Usage Example - Operation: 'DOC SEP'

Separate Document File into Multiple Document Files using Index from a doc file register

Procedure used: docsep.spi

.OPERATION: DOC SEP ;   Separate document file
.INPUT DOCUMENT FILE: sav_doc_sep ;   Doc file (input)
.REGISTER COL. USED FOR SEPARATION (0 IS KEY): 8 ;   Separation index register
.OUTPUT DOCUMENT FILE TEMPLATE: docsep_*** ;   Output doc file template (output)
.OPTIONAL DOCUMENT FILE HEADER: KEY PSI, THE, PHI ;   Doc file header comment
.SUMMARY DOCUMENT FILE: doc_sep_summary ;   Summary doc file (output)


INPUT DOC FILE
sav_docsep.jpg


FIRST OUTPUT DOC FILE
docsep_001
SECOND OUTPUT DOC FILE
docsep_002
THIRD OUTPUT DOC FILE
docsep_003
SUMMARY OUTPUT DOC FILE
docsep_summary
