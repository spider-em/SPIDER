Usage Example - Operation: 'RT SF '

Rotate & Shift an Image Using Fourier Based Spline Interpolation

Procedure used: rtsf.spi

.OPERATION: RT SF ;   Rotate & shift
.INPUT FILE NAME OR TEMPLATE: sav_rib_avg ;   File name (input)
.OUTPUT FILE: rtsf ;   Rotated file name (output)
.ROTATION ANGLE, SCALE: 161 , -1 ;   Rotation angle and scale
.SHIFTS IN X & Y: 5.87, -1.42 ;   X and Y shift


INPUT IMAGE OUTPUT IMAGE
sav_rib_avg rtsf
