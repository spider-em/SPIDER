Usage Example - Operation: 'PO'

Convert Image to Polar Coordinate Representation

Procedure used: po.spi

.OPERATION: PO ;   Convert to polar coordinates representation
.INPUT FILE: sav_pp ;   File name (input
.OUTPUT FILE: po ;   File name (output)
.INNER AND OUTER RADIUS: 5, 50 ;   Inner and Outer radii
.FULL OR HALF CIRCLE (F/H): F ;   Convert Full or Half circle


INPUT OUTPUT
sav_pp po
