Usage Example - Operation: NORM

Normalization of image/volume using average and deviation ||

Procedure used: ./images/norm.spi

.OPERATION: NORM [no],[avg],[std] ;   Normalization using average and deviation
.INPUT FILE NAME OR TEMPLATE: unnorm_0 ;   Un-normalized image (input)
.MASK (* IF NO MASK) FILE: norm_mask ;   Mask image
.OUTPUT FILE NAME OR TEMPLATE: norm_m_n ;   Normalized image (output)
.APPLY FLAT-FIELD RAMP (Y,N): N ;   Do not flat-field


INPUT IMAGE
unnorm_0
INPUT MASK
norm_mask


OUTPUT IMAGE
(No mask - No ramp)
norm_n_n
OUTPUT IMAGE
(No mask - Ramp)
norm_n_y
OUTPUT IMAGE
(Masked - No ramp)
norm_m_n
OUTPUT IMAGE
(Masked - Ramp)
norm_m_y
