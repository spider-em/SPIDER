Usage Example - Operation: 'CP to MRC'

Copy SPIDER file To MRC file

Procedure used: cptomrc.spi

.OPERATION: CP TO MRC ;   Copy SPIDER file To MRC file
.SPIDER INPUT FILE NAME OR TEMPLATE (E.G. SPI_STK@**): savpad_nik_sli_45 ;   SPIDER image (input)
.MRC OUTPUT FILE NAME OR TEMPLATE (E.G. **@MRC_STK.mrc): cptomrc.mrc ;   MRC image (output)
.MRC DATA MODE (0/1/2/6): 2 ;   mode 2 = 32 bit output
.DATA ORIGIN (UL/LL): UL: UL ;   Upper Left data origin
.PIXEL SIZE FOR (A) FOR X & Y AXES: 12, 12 ;   Pixel size


INPUT IMAGE
savpad_nik_sli_45
OUTPUT IMAGE
cptomrc.mrc
