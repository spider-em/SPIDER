Usage Example - Operation: 'TH F'

Threshold - Set Pixels Above/Below Threshold to Fixed Value

Procedure used: thf.spi

.OPERATION: TH F ;   Threshold - Set Pixels Above/Below Threshold to Fixed Value
.INPUT FILE: sav_pp ;   File name (input)
.OUTPUT FILE: thf ;   File name (output)
.ALTER (A)BOVE, (B)ELOW OR (C) BOTH SIDES (A/B/C) b ;   Change pixels above, below, or on boths sides of the threshold
.THRESHOLD: 170 ;   Set threshold
.FIXUP DENSITY: 0 ;   New Pixel value


INPUT OUTPUT
sav_pp thf
