Usage Example - Operation: 'CP FROM PDB'

Create SPIDER volume from PDB file

Procedure used: cpfrompdb.spi

.OPERATION: CP FROM PDB ;   Create SPIDER volume from PDB file
.PDB INPUT FILE: sav_efg_vol.pdb ;   PDB volume (input)
.VOXEL SIZE [A]: 1 ;   1 angstrom
.CENTER? (Y/N) : Yes ;   Center it
.ATOMS OR TEMPERATURE? (A/T): Atoms ;   Show atoms
.SPIDER VOLUME SIZE, NX, NY, & NZ: 109 109 109 ;   Dimensions
.SPIDER OUTPUT FILE: cpfrompdb_efg_vol ;   SPIDER volume (output)


OUTPUT VOLUME
(Surface)
cpfrompdb_efg_vol_surf
