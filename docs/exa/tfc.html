Usage Example - Operation: 'TF C'

Transfer Function - Generate a straight, complex, CTF correction image

Procedure used: ./images/tfc.spi

.OPERATION: TF C ;   Generate straight, complex CTF
.CTF OUTPUT FILE: tfct ;   CTF correction file (output)
.SPHERICAL ABBERATION CS [mm]: 2.0 ;   CS
.DEFOCUS [A], ELECTRON VOLTAGE [Kev]: 23400, 300 ;   Defocus, electon voltage
.NUMBER OF SPATIAL FREQUENCY POINTS IN X & Y: 360, 360 ;   Number of spatial freq. points
.MAXIMUM SPATIAL FREQUENCY [1/A]: 0.373 ;   Max spatial freq.
.SOURCE SIZE [1/A], DEFOCUS SPREAD [A]: 0.0, 0.0 ;   Source size, defocus spread
.ASTIGMATISM [A], AZIMUTH [DEG]: 0,0 ;   Astigmatism, azimuth
.AMPLITUDE CONTRAST RATIO [0-1], GAUSSIAN ENVELOPE HALFWIDTH: 0.1, 10000 ;   ACR, GEH (10000 = has no effect)
.SIGN [+1 or -1]: -1 ;   Sign
