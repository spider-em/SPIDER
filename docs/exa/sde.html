Usage Example - Operation: SD E

Close Document file

Procedure used: sde.spi

.OPERATION [x] = 30 ;   Set [x] to: 30
.OPERATION [y] = 45 ;   Set [y] to: 45
.OPERATION [z] = 60 ;   Set [z] to: 60
.OPERATION [c] = 100 ;   Set [c] to: 100
..  
.OPERATION SD 1,[x],[y] ;   Save register variables in doc. file
.DOCUMENT FILE sde_doc ;   Existing doc file name
..  
.OPERATION SD 2,[z],[c] ;   Save register variables in doc. file
.DOCUMENT FILE sde_doc ;   Existing doc file name
..  
.OPERATION SD E ;   Finished with this doc. file
.DOCUMENT FILE sde_doc ;   Existing doc file name


OUTPUT DOC FILE
sde_doc
