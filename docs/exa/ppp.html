Usage Example - Operation: 'PP P'

Set Pixel values in image/volume from Document file, alters neighbors

Procedure used: ppl.spi

X & Y factors
.OPERATION: PP P ;   Set Pixel value
.INPUT FILE: rib_avg ;   Image (input/output)
.DOCUMENT FILE: ppl_doc ;   Document file (input)
.X-COL, Y-COL: 1, 2 ;   X & Y registers in doc file
.INTENSITY COL.: 3 ;   Density register in doc file
.X-FACTOR, Y-FACTOR: 1,1 ;   Factors
.KEYS TO BE USED: 1-16 ;   Keys


INPUT DOCUMENT FILE
ppl_doc


INPUT IMAGE
OUTPUT IMAGE
rib_avg ppp
