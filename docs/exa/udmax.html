Usage Example - Operation: 'UD MAX'

Unsave Document - get max/min from document column

Procedure used: udmax.spi

.DOCUMENT FILE: sav_ang_doc ;   Doc file (input)
.REGISTER COLUMN TO BE SEARCHED (0 = KEY): 2   Column number
.

.
