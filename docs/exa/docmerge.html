Usage Example - Operation: 'DOC MERGE'

Merge Two Document Files

Procedure used: docmerge.spi

.OPERATION: DOC MERGE ;   Merge two document files
.FIRST DOCUMENT FILE: sav_docmerge_0 ;   First doc file (input)
.SECOND DOCUMENT FILE: sav_docmerge_1 ;   Second doc file (input)
.OUTPUT DOCUMENT FILE: docmerge_doc ;   Output doc file (output)
.COLUMN TO BE MERGED BY (0 IS KEY): 0 ;   Merge by keys


FIRST INPUT DOC FILE SECOND INPUT DOC FILE
sav_docmerge_0 sav_docmerge_1


OUTPUT DOC FILE
docmerge_doc
