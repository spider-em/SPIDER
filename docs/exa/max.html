Usage Example - Operation: 'MA X'

Mask image/volume in X direction

Procedure used: max.spi

.OPERATION: MA X ;   Mask image in x direction
.INPUT FILE: sav_pp ;   File name (input)
.OUTPUT FILE: max ;   File name (output)
.OUTER & INNER HALFWIDTH EXTENTS: 11 ;   Outer extent halfwidth of mask
.DISK, COSINE, GAUSSIAN EDGE OR TRUE GAUSSIAN (D/C/G/T): D ;   Mask type
.AVG, PREV AVG, CIRCUMF, OR EXTERNAL BACKGROUND? (A/P/C/E): PREV ;   Background type
.X CENTER OF MASK OR <CR&gr;FOR IMAGE CENTER: 40 ;   Center of mask


INPUT IMAGE OUTPUT IMAGE
sav_pp max
