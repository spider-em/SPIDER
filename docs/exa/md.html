Usage Example - Operation: 'MD'

Set Modes or options for operations

Procedure used: md.spi

.OPERATION: MD ;   Set mode
.MODE: ME ;   Lists the mode menu
.  
.OPERATION: MD ;   Set mode
.MODE: SET SEED ;   Sets seed for the F90 RANDOM_NUMBER generator
.VALUE: 12345 ;   Seed
.  
.OPERATION: MD ;   Set mode
.MODE: SET MP ;   Set number of OMP Parallel threads
.NUMBER OF PROCESSORS WANTED (OR 0 FOR ALL):4 ;   Number of OMP threads
.  
.OPERATION: MD ;   Set mode
.MODE: SET THREADS ;   Set number of FFTW threads
.NUMBER OF THREADS WANTED (OR 0 FOR ALL): 0 ;   Use all available FFTW OMP threads
.  
.OPERATION: MD ;   Set mode
.MODE: VB OFF ;   Shorten listing to RESULTS file
.  
.OPERATION: MD ;   Set mode
.MODE: FBS ON ;   Use Fourier Based Spline Interpolation in 'AP' routines
.  
.OPERATION: MD ;   Set mode
.MODE: STA ;   Gives the current status of 'MD' options
.  
.OPERATION: MD ;   Set mode
.MODE: VB ON ;   Verbose listing in RESULTS file
.  
.OPERATION: MD ;   Set mode
.MODE: RESULTS OFF ;   Terminates current SPIDER RESULTS file
.  
.OPERATION: MD ;   Set mode
.MODE: RESULTS ON ;   Restarts use of current SPIDER RESULTS file
.  
.OPERATION: MD ;   Set mode
.MODE: TERM ON ;   Diverts SPIDER RESULTS file to Terminal
.  
.OPERATION: MD ;   Set mode
.MODE: TERM OFF ;   Resets SPIDER RESULTS file output
.
