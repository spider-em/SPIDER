Usage Example - Operation: 'CTF ED'

Contrast Transfer Function - Estimate Defocus from micrograph

Procedure used: ctfed.spi

.OPERATION: CTF ED [def],[cutoff] ;   Estimate Defocus
.MICROGRAPH IMAGE FILE: sav_mic006_dc_2 ;   Micrograph image (input)
.TILE SIZE, X & Y TILE PERCENT OVERLAP: 500, 50, 50 ;   Tile size, x & y tile % overlap
.X & Y TILING BORDER: 500, 500 ;   X & Y tiling border
.PIXEL SIZE [A] & SPHERICAL ABBERATION CS [MM]: 2.8, 2 ;   Pixel size, Spherical aberration
.ELECTRON VOLTAGE [Kev]: 200 ;   Electron voltage
.AMPLITUDE CONTRAST RATIO [0-1]: 0.1 ;   Ampl. contrast ratio
.DEFOCUS NOISE DOC: ctfed_noise ;   Noise doc file (output)
.OUTPUT DEFOCUS DOCUMENT FILE: ctfed_defocus ;   Defocus doc file (output)
.KEY/IMAGE NUMBER FOR DOCUMENT FILE: 1 ;   Key/image number for doc file
.AVERAGE POWER SPECTRUM FILE: ctfed_pw_avg ;   2D power spectrum (output)

(Portion only)

ctfed_pw_avg

ctfed_defocus
