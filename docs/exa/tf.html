Usage Example - Operation: 'TF'

Transfer Function - Generate real CTF file, Defocus dependent

Procedure used: ./images/tfd.spi

.OPERATION: TF ;   Transfer Function - Defocus dependent
.OUTPUT FILE: tf ;   Real space CTF correction file (output)
.SPHERICAL ABBERATION CS [MM]: 2.0 ;   CS
.LAMBDA [A]: 0.0197 ;   Wavelength lambda
.LOWER DEFOCUS [A]: 3660 ;   Lower defocus
.UPPER DEFOCUS [A]: 10000 ;   Upper defocus
.NUMBER OF SPATIAL FREQ. POINTS & DEFOCUS POINTS: 64, 200 ;   Number of spatial frequency points
.MAXIMUM SPATIAL FREQUENCY [A-1]: 0.427 ;   Maximum spatial frequency
.SOURCE SIZE [A-1]: 0.005 ;   Source size
.DEFOCUS SPREAD [A]: 0.0 ;   Defocus spread
.AMPLITUDE CONTRAST RATIO [0-1], GAUSSIAN ENVELOPE HALFWIDTH [1/A]: 0.1 , 10000 ;   ACR, GEH
.DIFFRACTOGRAM / ENVELOPE / STRAIGHT? (D/E/S): S ;   Straight
.FRAME (Y/N): Y ;   Want frame


OUTPUT FILE
(Displayable)
tf
