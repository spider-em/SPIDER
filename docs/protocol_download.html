Procedure files from Nature Protocols article

The procedure and data sets referenced in:
Shaikh TR, Gao H, Baxter WT, Asturias FJ, Boisset N, Leith A, Frank J.
SPIDER image processing for single-particle reconstruction of biological macromolecules from electron micrographs.
Nature Protocols 3, 1941-1974 (2008).

Can be downloaded from: zenodo.org/records/14751455.
You will need the following data sets:

File (Mbytes)
reference_based.tar.gz 791
commonlines.tar 18
randomconical.tar 3

Uncompress and unpack the data archives:
tar -zxvf reference_based.tar.gz
tar -xvf commonlines.tar
tar -xvf randomconical.tar
