Icosahedral Reconstructions

in SPIDER




  • The reference structure has to be placed such that the 5-fold symmetry axis is on Z-axis.

  • Any refinement program (from Three Dimensional Reconstruction of Single Particle Specimens using Reference Projections) can be used with the following modifications:

    • Alignment operations should not check mirrored positions (see AP SH)

    • VO EA operation that appears in these procedures has to be modified such that the unique triangle specified is:
      theta = 0,37
      phi = 0,71.999

  • The 3D reconstruction operations need a icosahedral symmetry document file which is given below. Here is the same file for downloading: ico_symm.txt
