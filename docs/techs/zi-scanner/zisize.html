Advantages:

ZI.tif files contain data at a range of resolutions. Data can be screened at 5.64 A pixelsize. Later, data can be reconstructed at higher resolution.

SPIDER files are already converted. This speeds up Power_Spectra and Particle_Picking procedure files.

General rules:

1) Use appropriate scan resolution (i.e., smallest possible).
2) Extract the appropriate Spider overview.
3) Keep the smaller format (ZI.tif or Spider) on disk.
4) Archive to tape - don't use computer memory for long-term storage.
