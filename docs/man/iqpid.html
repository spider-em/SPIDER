IQ PID - Inquire current Process ID

(9/23/06)

PURPOSE

To obtain current process id number.   Example.

USAGE

.OPERATION: IQ PID [pid]
[Current process ID is printed out to results file.]

NOTES

  1. If a register variable is supplied on the operation line, the variable will be set to the current process ID.

  2. Implemented by: ArDean Leith
