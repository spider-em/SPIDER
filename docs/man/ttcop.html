TT COPY - Copy image Title

(6/04/99)

PURPOSE

Copy an image title to another existing file.   Example.

SEE ALSO

TT [Set image Title]

USAGE

.OPERATION: TT COPY

.TITLE SOURCE FILE: PIC001
[Enter the filename of the file that contains the title.]

.OUTPUT FILE: PIC002
[Enter filename of file to be altered.]

NOTES

  1. The title is usually transferred automatically to derivative files also.

SUBROUTINES

TITLE, LUNSETHDR
