AD - ADd images

(12/21/93)

PURPOSE

Adds two or more pictures or volumes together. The sum is kept in the first input file.

SEE ALSO

USAGE

.OPERATION: AD

.INPUT1 FILE: PIC001
[Enter the name of the first file to be added.]

.INPUT2 FILE: PIC002
[Enter the name of the second file to be added.]

In any event, the system continues:

.INPUT FILE: PIC003
Enter the name of the third file to be added.]

ADd will continue in this manner until an asterisk ('*') is given as the name of the input file.

NOTES

  1. The original contents of the first input file will be destroyed, since it is replaced by the sum of the pictures.

  2. ADD only works for files with the same dimensions. To add pictures of different dimensions, use PATCH (see 'PA').

  3. For adding an entire file series, is is more practical to use operation 'AS'. This will also produce a variance map.
