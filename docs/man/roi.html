RO I - Rotational average - 2D Image output

(03/25/96)

PURPOSE

Computes the radial distribution function of a two-dimensional density distribution stored in a square array. Center assumed at (NX/2+1, NY/2+1).   Example.

SEE ALSO

RO SD [Rotational average - single line output, doc file]

USAGE

.OPERATION: RO I

.INPUT FILE: PIC001
[Enter the name of the file to be rotationally averaged.]

.OUTPUT FILE: ROT001
[Enter the name of the file to store averages.]

[The output file will contain the azimuthally averaged density distribution as a function of radius. The dimensions of the output file will be identical to input image.]

SUBROUTINES

RADAVI, CRCSE2

CALLER

UTIL3
