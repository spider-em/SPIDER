SO C - Surface Comparision

(03/7/00)

PURPOSE

Finds statistics for comparison of two surfaces generated by the surface voxel extraction "SO" operation. Pixels/voxels which have an value of >= 1.0 are assumed to be on the surface.

SEE ALSO

IA [Interface Area]
SO [Surface extraction]

USAGE

.OPERATION: SO C

.FIRST INPUT VOLUME FILE: VOL000
[Enter the name of the image or volume.]

.FIRST INPUT SURFACE FILE: SURF000
[Enter the name of the image or volume containing surface voxels.]

.SECOND INPUT VOLUME FILE: VOL001
[Enter the name of the image or volume.]

.SECOND INPUT SURFACE FILE: SURF001
[Enter the name of the image or volume containing surface voxels.]

NOTES

  1. Following Statistics are Printed:

  2. Total Voxels on First Surface:
    Total Voxels on 2nd Surface:
    Total Number of Distances:
    Mean Inter-surface Distance:
    S.D. of Distances:
    Summed Sign of Density Difference:

  3. All files must be different as they are opened simultaneously.
