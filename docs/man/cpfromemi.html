CP FROM EMI - Copy EMISPEC format file to SPIDER file

(3/5/02)

PURPOSE

Copies some EMISPEC format files to a SPIDER format image file.

USAGE

.OPERATION: CP FROM EMI

.EMISPEC INPUT FILE NAME: CCP4IN
[Enter the name of the EMISPEC file you want to copy.]

.OUTPUT FILE: SPI000
[Enter the name of the SPIDER output file.]

NOTES

  1. Only works for EMISPEC 1026 x 1026 real mode files.

  2. Only the size from the CCP4 file header is utilized. Other CCP4 header info is discarded.

  3. This routine has not been tested for converting between other sizes. It may not work.

SUBROUTINES
