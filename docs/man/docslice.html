DOC SLICE - Put slice from document file into new document file

(4/26/16)

PURPOSE

Put slice from document file into new document file.   Example.

SEE ALSO

SD [Save Document]
UD [Unsave Document]
DOC CREATE [Create Document file]
DOC MERGE [Merge two Document files]
DOC REN [Renumber Document file keys]

USAGE

.OPERATION: DOC SLICE

.INPUT DOCUMENT FILE: DOC001
[Enter name of input document file.]

.OUTPUT DOCUMENT FILE: DOC002
[Enter name of output document file.]

.KEYS TO PLACE IN NEW DOC FILE: 5-7,9
[Enter numbers of keys from input document file whose registers are to be transfered to output document file.]

NOTES

  1. Key numbers are renumbered consecutively in output document file.

SUBROUTINES

DOCS1, OPENDOC, LUNDOC

CALLER

DRIV1
