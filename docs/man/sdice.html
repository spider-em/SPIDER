SD ICE - Save Document - In Core, End

(04/01/13)

PURPOSE

'SD ICE' is used after 'SD IC ' or 'UD IC' to terminate access to the current in-core document file (array). This operation duplictes action of 'UD ICE'.   Example.

SEE ALSO

UD IC [Unsave Document - In-core, Retrieve register values]
UD ICE [Unsave Document - End use of In Core document file]

USAGE

.OPERATION: SD ICE
   <or>
SD IC END

.DOCUMENT FILE: DOC000
Terminates in-core use of DOC000 document file.]

SUBROUTINES

UDOCIC, SDOCIC

CALLER

DRIV3
