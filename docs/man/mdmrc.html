MD MRC - Set Modes or options for MRC related operations

(11/21/19)

PURPOSE

Switches between different modes for MRC file I/O.

SEE ALSO

MD [Set Modes or options for operations]

USAGE

.OPERATION: MD MRC

.MRC MODE (AXIS/MENU/STATUS): AX
[Enter the mode/option. The available choices are: 
ME Lists this mode menu.
STA Gives the current status of 'MD MRC' options.
AXIS Sets default origin and handedness for new MRC files.

For "AX" SPIDER will ask:


.DATA ORIGIN (UL/LL): UL
[Origin options are 'Upper left' or 'Lower left'. This is the pixel location which occurs first in the MRC files data content]

.DATA HANDEDNESS (L/R)':
[Handedness options are 'Left' or 'Right'. This determines the slice of the volume which occurs first in the MRC files data content.]

NOTES

  1. The origin and handedness is NOT defined by any MRC file standard. It varies by software! Thus display of a MRC image with software other than that from a single package may yield mirrored result.

SUBROUTINES

SETMODE, SETMODE_MRC
