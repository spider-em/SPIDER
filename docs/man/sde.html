SD E - Save Document - End

(9/20/96)

PURPOSE

Closes a specified document file.   Example.

SEE ALSO

SD [Save Document]

USAGE

.OPERATION: SD E

.DOCUMENT FILE: DOC001
[Enter the name of the file. This document file will be closed and removed from the list of current document files.]

SUBROUTINES

SAVDOCQ, OPENDOC

CALLER

DRIV3
