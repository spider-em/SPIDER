MRC HE - List values in MRC data file header

(10/27/19)

PURPOSE

List values in a MRC file header.   Example.

SEE ALSO

ST H [Set values in image/volume Header by name]
FI X [File Information - Retrieve file header value(s) by location]

USAGE

.OPERATION: MRC HEAD

.MRC FILE: 22sav_mo_stk.mrc
[Enter the name of the MRC file including extension.]
