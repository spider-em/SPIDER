DE - Delete a file

(12/31/93)

PURPOSE

Deletes a file. Can delete single stacked SPIDER file also.   Example.

SEE ALSO

DE A [Delete file series]

USAGE

.OPERATION: DE

.DELETE FILE: PIC001
[Enter the name of the file to be deleted. If the file exists, it will be deleted. The message "DELETED: PIC001" will be printed on the terminal.]
