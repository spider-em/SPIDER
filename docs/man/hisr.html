HIS R - Histogram - Image/volume, Restricted range

(8/12/13)

PURPOSE

Computes histogram of image/volume and outputs to desired file. Only original pixel/voxel values within specified range will be included in the histogram.]   Example.

SEE ALSO

HIS [Histogram - image/volume]

USAGE

.OPERATION: HIS R

.INPUT FILE: IMG027
[Enter the name of the file containing the image/volume.]

.MASK FILE (* IF NONE): MASK027
[Enter the name of file containing a mask for histogram area. Pixels//voxels containing values > 0.5 are included in histogram. Enter '*' if no masking is wanted.]

.OUTPUT TO RESULTS FILE, DOC. FILE, OR TERMINAL? (R/D/T): R
[ Enter destination type for histogram.]

If output is sent to a document file SPIDER asks:


.DOCUMENT FILE: DOC000
[Enter the name for the document file that gets the histogram output.]

.NUMBER OF HISTOGRAM BINS: 30

.HISTOGRAM RANGE (MIN ... MAX): 0.1, 1.2
[Only original pixel/voxel values within this range will be included in the histogram.]

[A histogram of the image values containing the specified number of bins will be computed and put into the destination file. In addition the following statistics will be given:


FILE RANGE: .........
HISTOGRAM RANGE: .........
FILE PIXELS: HIST. PIXELS:
NO. OF BINS: BIN SIZE:
HIST. MEAN: HIST. MODE:
HIST. S.D.:
