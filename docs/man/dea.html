DE A - Delete file series

(08/10/98)

PURPOSE

Deletes a file series.   Example.

SEE ALSO

DE [Delete a file]

USAGE

.OPERATION: DE A

.FIRST FILE: PIC001
[All files with the prefix PIC, starting with PIC001, will be deleted. A gap of up to 10 file numbers will be searched before the deleting stops. The message of searching and deleting will be printed on the terminal.]
