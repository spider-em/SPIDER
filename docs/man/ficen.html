FI CEN - File Information - report image/volume center

(12/12/14)

PURPOSE

Find image/volume center using SPIDER's convention and and optionally place center values in specified register variables.   Example.

SEE ALSO

ST H [Set values in image/volume Header by name]
ST [Set values in data file header]

USAGE

.OPERATION: FI CEN [xc],[yc],[zc]
[Find image/volume center. This operation can return up to 3 optional register variables which contain:
Variable Example Receives
First [xc] X center coordinate
Second [yc] Y center coordinate
Third [zc] Z center coordinate

.INPUT FILE: PIC002
[Name of input file.]

SUBROUTINES

FINDFILCEN

CALLER

UTIL1
