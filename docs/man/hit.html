HI T - Histogram - output on Terminal

(8/10/98)

PURPOSE

Computes and prints out the histogram of an image on the terminal.   Example.

SEE ALSO

HI R [HIstogram - output in Results file, use specified range.]
HI R [HIstogram - output in Results file, use specified range.]
HI TR [Histogram - output on Terminal, use specified range]
HI D [Histogram - output in Document file]
HI DR [Histogram - output in Document file, use specified Range]
HI M [Histogram - within Masked area]

USAGE

.OPERATION: HI T

.INPUT FILE: PIC027
[Enter the name of the file containing the image.]

A 70 bin histogram of the image values will be computed and output to the terminal. The following information will also be printed:

FILE RANGE: .........
HISTOGRAM RANGE: .........
FILE PIXELS: HIST. PIXELS:
NO. OF BINS: BIN SIZE:
HIST. MEAN: HIST. MODE:
HIST. S.D.:

NOTES

  1. The histogram is no longer permanently kept in the file header (1995).
