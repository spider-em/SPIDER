CP TO VOL - Copy images into volume

(5/21/13)

PURPOSE

Copy a series of images (not volumes) into a specified slices of a volume. If the output volume already exists replaces the specified slices.   Example.

SEE ALSO

SK [Stack images/volumes into volume]
SK R [Replace a slice in volume]
PS Z [Pick Slice - Z (xy planes)]

USAGE

.OPERATION: CP TO VOL

.INPUT FILE NAME OR TEMPLATE (E.G. STK@****): SER***
[Enter template for file of the series to be stacked.]

If the input file name is a template, SPIDER asks:

.FILE NUMBERS OR SELECTION DOC. FILE NAME: select_03
[Enter the file numbers to be substituted into the file name template or name of a selection document file that contains the input file numbers in the first register.]

.OUTPUT VOLUME FILE: VOL001
[Enter the name of the volume to hold the slices.]

.OUTPUT SLICE NUMBERS: 1-5
[Enter list of slices that will be filled in the output file.]

NOTES

  1. Does not stack volumes into a new volume.
