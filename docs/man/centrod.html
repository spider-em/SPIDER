CENT ROD - Center image(s) containing rod ||

(1/29/12)

PURPOSE

Rotates, and shifts image that contains a linear object such as a rod shape so as to align rod vertically in center of image. Output image has SAME size as input image.   Example.

SEE ALSO

CG [Center of Gravity]
CG PH [Center of Gravity - Phase approximation ||]
RT SF [Rotate and Shift - image, Fourier based spline interpolation ||]

USAGE

.OPERATION: CENT ROD

.INPUT FILE NAME OR TEMPLATE (E.G. STK@****): instack03@
[Enter the name of the file to be rotated. This file can be a single file, a whole stack (ending in @), or a template for a set of images (e.g stack3@****** or fil***).]

If the input file is a single image SPIDER asks:


.OUTPUT FILE: IMG048
[Enter the name of the file where the aligned image is to be placed.]

If the input file is a whole stack SPIDER asks:


.OUTPUT STACK FILE: STACK03@
[Enter the name of the file where the aligned images are to be placed. This output file will be a whole stack.]

.ALIGNMENT DOCUMENT OUTPUT : rodaligndoc
[Enter the name for a document file where the rotation and shift parameters will be placed. Enter '*' if you do not want such this file.]

If the input file is a set of images SPIDER asks:


.FILE NUMBERS OR SELECTION DOC. FILE NAME: select_03
[Enter the file numbers to be substituted into the file name template or name of a selection document file that contains the file numbers in the first register.]

.OUTPUT FILE TEMPLATE: STACK03@****
[Enter the name of the file(s) where the rotated and shifted images are to be placed.]

.ALIGNMENT DOCUMENT OUTPUT : rodaligndoc
[Enter the name for a document file where the rotation and shift parameters will be placed. Enter '*' if you do not want such this file.]

NOTES

  1. Does not work on volumes.

  2. The accuracy of interpolation is high due to use of Fourier based bicubic spline which is also used in: 'RT SF'.

  3. Implemented by: G. Kishchenko.
