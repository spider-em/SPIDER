HIS - Histogram - image/volume

(8/12/13)

PURPOSE

Compute histogram of image/volume and output to desired file.   Example.

SEE ALSO

HIS R [Histogram - Image/volume, Restricted range]

USAGE

.OPERATION: HIS

.INPUT FILE: IMG027
[Enter the name of the file containing the image/volume.]

.MASK FILE (* IF NONE): MASK027
[Enter name of file containing a mask for histogram area. Pixels/voxels containing values > 0.5 are included in histogram. Enter '*' if no masking is wanted.]

.OUTPUT TO RESULTS FILE, DOC FILE, OR TERMINAL? (R/D/T): R
[Enter destination type for histogram.]

If output is sent to a document file SPIDER asks:


.DOCUMENT FILE: DOC000
[Enter the name for the document file that gets the histogram output.]

.NUMBER OF HISTOGRAM BINS: 30
[Enter the number of bins used in the histogram.]

[The histogram will be computed and placee in the destination file. In addition the following statistics will be given:

FILE RANGE: .........
HISTOGRAM RANGE: .........
FILE PIXELS: HIST. PIXELS:
NO. OF BINS: BIN SIZE:
HIST. MEAN: HIST. MODE:
HIST. S.D.:

SUBROUTINES

HISTOG, GRAPHS

CALLER

UTIL1
