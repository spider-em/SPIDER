LO LSD - Prepare LSD volume for Locating motif inside volume

(8/04/2008)

PURPOSE

Prepare LSD volume file for motif search. This file is needed during motif location using operation: 'LO'.

SEE ALSO

LO I [Initialize files for Locating motif inside volume]
LO [Locate motif inside a search volume]

USAGE

.OPERATION: LO LSD

.ROTATED MOTIF MASK INPUT FILE: rot_mask
[Enter name for rotated mask for motif (usually created by: 'LO I').]

.PADDED MOTIF MASK FFT INPUT FILE: mask_fft
[Enter name for FFT of mask (usually created by: 'LO I').]

.SEARCH VOLUME FFT INPUT FILE: search_fft
[Enter name for FFT of search volume (usually created by: 'LO I').]

.SEARCH VOLUME SQ. FFT INPUT FILE: search_sq_fft
[Enter name for FFT of search volume squared (usually created by: 'LO I').]

.LSD VOLUME OUTPUT FILE: lsd
[Enter name for local standard deviation file.]

NOTES

  1. The following procedures: new_sigs_settings.spi, new_sigs_pub.spi, and new_sigsloop.spi contain an example of this operation in use.
