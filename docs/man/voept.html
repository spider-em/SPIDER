VO EPT - Vector Operation - Euler angle point rotation

(08/04/04)

PURPOSE

Rotate a point using given Euler angles. Assumes SPIDER conventions for Eulerian angles.   Example.

SEE ALSO

VO IA [Vector Operation - Find Included Angle]

USAGE

.OPERATION: VO EPT [x],[y],[z]
[This operation can return up to three optional register variables:

Variable Example Receives
First [x] Rotated X coordinate
Second [y] Rotated YZ coordinate
Third [z] Rotated Z coordinate

.INITIAL POINT - X, Y & Y: 0, 0, 1
[Enter initial point (vector) (Use 0,0,1 for SPIDER projections).

.EULER ANGLES - PHI, THETA & PSI: 0, 25, 30
[Enter Euler angle for transformation.]
