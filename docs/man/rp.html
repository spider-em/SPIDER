RP - Replace Pixel value

(09/15/12)

PURPOSE

Replaces pixel value at a specified location in an image/volume.   Example.

SEE ALSO

GP [Get Pixel value]
PP L [Put Points in image - from Document file Listing]

USAGE

.OPERATION: RP

.INPUT FILE: PIC001
[Enter name of image/volume to be processed. This file will be altered.]

If the input file is a 2D image SPIDER asks:


.COLUMN & ROW: 21,56
[Enter X & Y coordinates of the pixel to be replaced.]

Alternatively if the input file is a 3D volume SPIDER asks:


.COLUMN, ROW, & SLICE: 21,56,32
[Enter X, Y, & Z coordinates of the voxel to be replaced.]

.NEW PIXEL VALUE: 1.23
[Enter pixel/volume density.]

NOTES

  1. It is no longer possible to specify the new pixel value in an operation line register.

  2. This operation has also been optimized for use with inline files.

SUBROUTINES

GPRP, OPENINLN

CALLER

UTIL2
