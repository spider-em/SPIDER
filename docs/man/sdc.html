SD C - Save Document - from CA S (_IMC) file

(9/6/09)

PURPOSE

Transfer image coordinates along factors from correspondence analysis coordinates file (_IMC00) into a document file. This document file can be used in WEB "CORR MAP" operation.

SEE ALSO

CA S [Correspondence Analysis - Solution]

USAGE

.OPERATION: SD C

.CORAN/PCA FILE PREFIX: coran_t_
[Enter the prefix for the files created in 'CA S'.]

.FACTORS NUMBERS (e.g. 1-8,12-22,55): 1-8,11-12
[Numbers are consecutive, and refer to the sequence of files specified in 'CA S'. Can specify up to 50 factors.]

.DOCUMENT FILE: DOC003
[Enter name for new document file.]

NOTES

  1. The coordinate n-tupels are transferred in the order the images were entered in 'CA S'. The image number is used as key.

  2. Example for retrieval of the position of image number 51 along factor 2:
    UD 51,[a1],[a2]
    DOC003
    After execution of this command, SPIDER register variable [a2] contains the desired value.

SUBROUTINES

RDCLUS, SAVD, OPAUXFILE, LUNDOC

CALLER

DRIV3
