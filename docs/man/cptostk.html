CP TO STK - Copy a series of stacks into a single new stack CP TO STK

(4/21/16)

PURPOSE

Copy a series of stacks (not images) into a specified stack. If the output stack already exists replaces the whole stack.   Example.

SEE ALSO

CP [Copy SPIDER file to SPIDER file]
CP TO VOL [Copy images into volume]

USAGE

.OPERATION: CP TO STK

.BARE STACK TEMPLATE (E.G. STK_***@): STK***@
[Enter template for the stack series to be merged.]

.FILE NUMBERS OR SELECTION DOC. FILE NAME: 1-20
[Enter the file numbers to be substituted into the file name template or name of a selection document file that contains the input file numbers in the first register.]

.INITIAL IMAGE NUMBER IN OUTPUT STACK: 1
[First image in new stack.]

.OUTPUT STACK (E.G. NEWSTK@): NEWSTK@
[Enter name for the new stack.]
