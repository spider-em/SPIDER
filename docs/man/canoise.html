CA NOISE - Correspondence Analysis - Create Eigenvalue doc file for noise

(9/10/13)

PURPOSE

Create document file containing eigenvalues from Coran or PCA analysis and the percentage of variance accounted for by each factor. This doc file can be used to determine which factors (vectors) are likely due to noise.
See: Classification and Clustering Tutorial for further info.   Example.

SEE ALSO

CA S [Correspondence Analysis - Solution]
SD C [Save Document - from CA S (_IMC) file]

USAGE

.OPERATION: CA NOISE [nfac]
[This operation can return one optional register variable:
Variable Example Receives
First [nfac] Number of factors

.CORAN/PCA FILE PREFIX: cas_ca
[Enter the prefix identifying the data files created from a previous 'CA S' run: _IMC, _PIX, & _EIG. Do not include trailing underscore.]

.EIGENVALUE DOC FILE: cas_ca_noise
[Enter name for document file that receives eigenvalues and percentages.]
