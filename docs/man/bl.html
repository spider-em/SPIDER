BL - Create Blank image/volume

(4/24/03)

PURPOSE

Creates an image/volume with a constant specified background.   Example.

SEE ALSO

MO [Create Model image]
MO 3 [Create Model volume]
PT [Create Pattern or mask image]

USAGE

.OPERATION: BL

.OUTPUT FILE: IMG014
[Enter the name of the file for the output image.]

.DIMENSIONS (NX,NY,NZ): 64,64,1
[Enter the dimensions of the output image/volume.]

.USE AVERAGE FROM ANOTHER IMAGE AS BACKGROUND? (Y/N): Y
[Enter whether the background should be taken as the average of another image, or should be entered by the user.]

If answer is "Y" - Yes. SPIDER asks:

.REFERENCE FILE: PIC002
[Enter the name of the picture the average of which is to be used as the background.]

The output will now be a picture of size 64x64, and it will contain a background that is the average of the reference file image.

If answer is "N" - No. SPIDER asks:

.BACKGROUND: 2.0
[Enter the desired background. In this case, the output image would be filled with pixels having the value 2.0.]
