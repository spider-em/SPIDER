MA L - Mask - line across image/volume

(9/23/96)

PURPOSE

Masks a specified image/volume with a line of specified background intensity.   Example.

SEE ALSO

MA [Mask - image/volume with a circular mask]
MA X [Mask - image/volume in X direction]
MA Y [Mask - image/volume in Y direction]
MM [Mask Multiplication]

USAGE

.OPERATION: MA L

.INPUT FILE: PIC001
[Enter name of the file to be masked.]

.OUTPUT FILE: IMG030
[Enter name of the output file.]

If the input file is a volume SPIDER asks:

.SLICE NUMBER: 12
[Enter the slice which has a line to be masked.]

.LINE NUMBER TO BE MASKED: 16
[Enter the line to be masked.]

.BACKGROUND: 0.0
[Enter the value to be put in the masked line.]
