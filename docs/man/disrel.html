DIS REL - Display image(s) using Relion_display

(1/8/20)

PURPOSE

Display SPIDER or MRC file using Relion_display. Requires that your system has Relion_display installed and accessible as: 'relion_display'.

SEE ALSO

DIS [Display single image using ImageMagick]

USAGE

.OPERATION: DIS REL

.SPIDER OR MRC INPUT FILE: PIC001
[Enter name of the SPIDER or MRC image/volume file.]

.RELION_DISPLAY OPTIONS (or <CR>): --scale .4
[Enter any valid option for the relion_display command.]

NOTES

  1. The origin and handedness of data in a MRC file is NOT defined by any MRC file. It varies by software! Thus display of a MRC image/volume from software other than Relion may yield a mirrored result.
