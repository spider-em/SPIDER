PS X - Pick Slice - X, (yz planes)

(1/27/96)

PURPOSE

Picks X slice out of a SPIDER volume file.   Example.

SEE ALSO

PS [Pick Slice - Y (xz planes)]
PS Z [Pick Slice - Z (xy planes)]
PS A [Pick Slice - Arbitrary direction]
WI [Window image/volume]

USAGE

.OPERATION: PS X
    <or>
PS XN

.INPUT FILE: SPH001
[Enter the name of the file containing the three-dimensional information.]

.OUTPUT FILE: SLI001
[Enter the name of the file where the two-dimensional slice (yz plane) is to be put.]

.X-SLICE NUMBER: 15
[Enter the number of the slice you wish to select.]

NOTES

  1. Due to the way the volume is stored on disk, 'PS' (='PS Y' by default) and 'PS Z' are much faster than 'PS X'.

  2. Use 'PS XN' to keep the FMIN and FMAX values from the original volumes in the slices.

SUBROUTINES

PICKSV

CALLER

VTIL2
