FT R - Fourier Transform Replace

(08/14/01)

PURPOSE

Synthesizes a Fourier transform from the amplitude of a given Fourier Transform I and the phase of another Fourier transform II.

SEE ALSO

FT [Fourier Transform ||]
FF [Filter - Fourier]
FP [Interpolate/Resize using Fourier transform ||]
FQ [Filter - Quick Fourier, High, low pass, etc., Padded ||]

USAGE

.OPERATION: FT R

.FIRST INPUT FILE: FOU001
[Enter file name of first Fourier transform.]

.SECOND INPUT FILE: FOU002
[Enter file name of second Fourier transform.]

.OUTPUT FILE: FOU003
[Enter output file name that will contain the new Fourier transform.]

NOTES

  1. This operation can be used to implement the "amplitude replace" step in a Gerchberg-Saxton or solvent flattening scheme.
