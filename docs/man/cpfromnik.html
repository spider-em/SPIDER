CP FROM NIKON - Copy Nikon Coolscan format file to SPIDER file

(1/13/05)

PURPOSE

Copies 16 bit integer Nikon Cooolscan 9000 TIFF input image to SPIDER file. May work for some other 16 bit TIFF images also.   Example.
