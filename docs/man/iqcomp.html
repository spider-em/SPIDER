IQ COMP - Inquire which compiler was used for this SPIDER

(4/22/13)

PURPOSE

Lists which compiler was used for compilation of the current SPIDER executable.   Example.

USAGE

.OPERATION: IQ COMP [compiler]

[The operation line can specify an optional register variable which receives The value

[Optional output register receives --number indicating which compiler was used.
0 If compiled with: PGFORTRAN compiler.
1 If compiled with: GFORTRAN compiler.
2 If compiled with: INTEL FORTRAN compiler.]

SUBROUTINES

INQUIRECOMP

CALLER

UTIL4
