VO IA - Vector Operation - Find Included Angle

(09/03/14)

PURPOSE

Find included angle (in degrees) between two vectors.   Example.

SEE ALSO

VO EPT [Vector Operation - Euler angle point rotation]
SA [Sum Alignment]
SA 3 [Sum Alignment in 3D]

USAGE

.OPERATION: VO IA [inc-ang]
[This operation can return one optional register variable:
Variable Example Receives
First [inc-ang] Included angle

.FIRST VECTOR - X, Y & Z: 1, 1, 1
[Enter X, Y, & Z components of first vector.]

.SECOND VECTOR - X, Y & Z: 1, -1, 0
[Enter X, Y, & Z components of second vector.
