PDB STP - PDB - Create PDB file from coordinate list

(1/11/14)

PURPOSE

Convert list of coordinates given in a SPIDER document file into a PDB file.   Example.

SEE ALSO

PDBCG3 [PDB - Calculate center of gravity]
PDBIF [PDB - Find information]
PDBPTS [PDB - Put coordinates in SPIDER document file.]
PDBRT3 [PDB - Rotate by Euler angles in 3D space]
PDBRT3A [PDB - Rotate by Euler angles around arbitrary center]
PDBRT3L [PDB - Rotate PDB file around line]
PDBSH [PDB - SHift coordinates]
UUSAGE:   PDB STP

          .SPIDER DOCUMENT FILE: spidoc
          [Enter the name of the SPIDER document file.] 

          .PDB OUTPUT FILE: abc.pdb
          [Enter the name of the output PDB file with extension 'pdb'.]    
			
NOTES:    1. Yields a standard format PDB file with  terminal atoms.  

          2. Implemented by Zhong Huang, Pawel A. Penczek.
