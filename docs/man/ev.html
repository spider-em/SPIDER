EV - Set system environmental variable

(8/13/14)

PURPOSE

Set system environmental variable from SPIDER operation line. Variable is visible within current SPIDER procedure. (Note that if a variable is set with 'SYS' operation it is not visable.) This operation is NOT AVAILABLE if SPIDER is compiled with gfortran.   Example.

SEE ALSO

SYS [System call]

USAGE

.OPERATION: EV

.VARIABLE: MY_IMG_LOC
[Name of system environmental variable to be set. Please note that environmental variables are case senstitive]

.VALUE: /mydir/imagedir/
[Value to be given to the system environmental variable. This is a string so register variables are not substituted.] E.g. if you give a value of: subdir{***[n]} then the value for the variable is exactly the same character sequence i.e.: subdir{***[n]}. Subsitution for:{***[n]} does not occur.

NOTES

  1. To access this environment variable within a string use ${MY_IMG_LOC}.
