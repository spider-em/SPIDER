UD ICE - Unsave Document - End use of In Core document file

(06/17/99)

PURPOSE

'UD ICE' is used after 'UD IC' or 'SD IC NEW' has created an in-core document file (array). This operation terminates access to the specified in-core document file (array).   Example.

SEE ALSO

UD IC [Unsave Document - In-core, Retrieve register values]
SD IC NEW [Save Document - In Core, create New storage array]

USAGE

.OPERATION: UD ICE

.DOCUMENT FILE: DOC000
Terminates in-core document file (array) use for the named file.]

SUBROUTINES

UNSDAL, LUNDOC

CALLER

DRIV3
