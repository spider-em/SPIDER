PJ 2 - Project - 2D volume onto line

(06/08/18)

PURPOSE

Computes a projection of a 2D density distribution onto a single line. Can project along X or Y axis.   Example.

SEE ALSO

PJ 3 [Project - 3D volume using Eulerian angles, trilinear or FBS interpolation, single projection]
PJ CYL [Project - 3D volume onto Cylinder]

USAGE

.OPERATION: PJ 3

.INPUT FILE: PIC001
[Enter the name of the image whose projection is to be computed.]

.PROJECT ALONG X or Y AXIS? (X/Y): Y
[ Projection direction.]

.OUTPUT FILE: LIN001
[Enter name for the output file. This will be a 1D SPIDER line image. In addition to this 1D image SPIDER will create a 2D visualization image with suffix '_vis' which can be easily viewed and processed further.]

SUBROUTINES

PJ2D

CALLER

VTIL2
