Usage Example - Operation: 'CTF FIND'

Contrast Transfer Function - Estimation of CTF parameters

Procedure used: ctffind.spi

.OPERATION: CTF FIND [def],[ang],[mag],[df1],[df2],[mrcang] ;   CTF Estimation
.INPUT IMAGE: sav_mic006_dc_2 ;   Micrograph image (input)
.SPIDER POWER SPECTRUM FILE: ctffind_pow_spi ;   SPIDER power spectrum (output)
.DIAGNOSTIC POWER SPECTRUM FILE: ctffind_pow_diag ;   Diagnostic power spectrum (output)
.OUTPUT DEFOCUS DOC FILE: ctffind_def ;   Defocus doc file (output)
.KEY FOR DEFOCUS DOC FILE: 1 ;   Key number
.SPHERICAL ABBERATION CS[mm], VOLTAGE[kV], & ACR: 2.0, 200, 0.1 ;   CS[mm], HT[kV], AmpCnst
.MAGNIFICATION, PIXEL SIZE[um], & BOX SIZE[pixels]: 50000, 14, 500 ;   XMAG, DStep[um], Box Size
.LOWER & UPPER RESOLUTION[A]: 35, 7.5 ;   ResMin[A], ResMax[A]
.LOWER DEFOCUS[A], UPPER DEFOCUS[A], & DEFOCUS STEP[A]: 10000, 40000, 5000 ;   dFMin[A], dFMax[A], FStep


INPUT MICROGRAPH
(Portion)
sav_mic006_dc_2


SPIDER POWER SPECTRUM IMAGE
( )
ctffind_pow_spi


DIAGNOSTIC POWER SPECTRUM IMAGE
( )
ctffind_pow_diag


OUTPUT DEFOCUS DOC FILE
ctffind_def


SPIDER RESULTS FILE
ctffind_resu
