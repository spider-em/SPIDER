Usage Example - Operation: 'TF DNS'

Transfer Function - Delete noise background

Procedure used: ./images/tfdns.spi

.OPERATION: TF DNS [mins] ;   Transfer Function - Delete noise background
.POWER SPECTRUM 1D IMAGE FILE: sav_rosdim ;   Rotational average - single line image (input)
.MAX SPATIAL FREQUENCY [A-1]: .20 ;   Max spatial frequency [A-1]
.SEARCH NEIGHBORHOOD DISTANCE: 20 ;   Search neighbor area
.CHANGE SEARCH NEIGHBORHOOD? (Y/N): No ;   Do not change search neighbors
.LIST OF MINIMA TO BE USED: 2-4 ;   List of minima to be used
.DENOISED PROFILE FILE: tfdns ;   Denoised rotational average (output)
.OUTPUT DOCUMENT FILE: tfdns_doc ;   Denoised rotational average doc file (output)

(PLOTTED)
sav_rosdim_plot

(Truncated)
tfdns_doc
