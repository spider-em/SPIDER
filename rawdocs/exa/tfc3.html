Usage Example - Operation: 'TF C3'

Transfer Function - Generate a straight, complex, CTF correction volume

Procedure used: ./images/tfc3.spi

.OPERATION: TF C3 ;   Generate straight, complex, CTF
.CTF OUTPUT FILE: tfct ;   CTF correction volume (output)
.SPHERICAL ABBERATION CS [mm]: 2.0 ;   CS
.DEFOCUS [A], ELECTRON VOLTAGE [Kev]: 3660, 300 ;   Defocus, electon voltage
.NUMBER OF SPATIAL FREQUENCY POINTS: 64 ;   Number of spatial freq. points
.MAXIMUM SPATIAL FREQUENCY [1/A]: 0.427 ;   Max spatial freq.
.SOURCE SIZE [1/A], DEFOCUS SPREAD [A]: 0.005, 0.0 ;   Source size, defocus spread
.ASTIGMATISM [A], AZIMUTH [DEG]: 2000, 0 ;   Astigmatism, azimuth
.AMPLITUDE CONTRAST RATIO [0-1], GAUSSIAN ENVELOPE HALFWIDTH: 0.1, 10000 ;   ACR, GEH (10000 = has no effect)
.SIGN [+1 or -1]: -1 ;   Sign


OUTPUT FILE
(Complex 3D = Not displayable)
tfc3
