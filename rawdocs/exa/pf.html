Usage Example - Operation: 'PF'

Profile an image row in a Postscript graph

Procedure used: pf.spi

.OPERATION: PF ;   Profile across an image file
.IMAGE: savavg000 ;   Image (input)
.POSTSCRIPT OUTPUT FILE: pf_plot ;   Postscript plot (output)
.INDIVIDUAL, GLOBAL, OR SUM SCALE PLOT? (I/G/S): I ;   Individual scaling
.ROW NUMBER(S): 20,32 ;   Rows
.NEW LOWER, UPPER AXIS BOUNDS OR <CR>: 0,65 ;   X axis
.NEW AXIS LABEL UNIT AND TICKS / LABEL OR <CR>: 10, 10 ;   Label unit & tick unit
.LABEL NO. TO EDIT... OR <CR> TO CONTINUE: <CR> ;   <CR>
.NEW AXIS LABEL UNIT AND TICKS / LABEL OR <CR>: -.09 0.1 ;   Y axis
.NEW LOWER, UPPER AXIS BOUNDS OR <CR>>: .02 5 ;   Label unit & tick unit
.LABEL NO. TO EDIT... OR <CR> TO CONTINUE: <CR> ;   <CR>
.PRINT NOW? (Y/N): No ;   No print


INPUT IMAGE
savavg000

PROFILE PLOT
pf_plot.ps
